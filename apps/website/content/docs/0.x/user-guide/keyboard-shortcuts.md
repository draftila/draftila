# Keyboard Shortcuts

## Tools

| Shortcut       | Tool                            |
| -------------- | ------------------------------- |
| `V`            | Move                            |
| `H`            | Hand (pan)                      |
| `C`            | Comment                         |
| `F`            | Frame                           |
| `R`            | Rectangle                       |
| `O`            | Ellipse                         |
| `Y`            | Polygon                         |
| `S`            | Star                            |
| `L`            | Line                            |
| `A`            | Arrow                           |
| `T`            | Text                            |
| `P`            | Pen                             |
| `Shift` + `P`  | Pencil                          |
| `N` or `Enter` | Node editing (on selected path) |

## Selection

| Shortcut         | Action                    |
| ---------------- | ------------------------- |
| `Cmd/Ctrl` + `A` | Select all                |
| `Escape`         | Deselect all / exit group |
| `Tab`            | Select next sibling       |
| `Shift` + `Tab`  | Select previous sibling   |

## Clipboard

| Shortcut                   | Action         |
| -------------------------- | -------------- |
| `Cmd/Ctrl` + `C`           | Copy           |
| `Cmd/Ctrl` + `X`           | Cut            |
| `Cmd/Ctrl` + `V`           | Paste          |
| `Cmd/Ctrl` + `Shift` + `V` | Paste in place |
| `Cmd/Ctrl` + `D`           | Duplicate      |
| `Cmd/Ctrl` + `Alt` + `C`   | Copy style     |
| `Cmd/Ctrl` + `Alt` + `V`   | Paste style    |
| `Delete` / `Backspace`     | Delete         |

## Transform

| Shortcut             | Action          |
| -------------------- | --------------- |
| Arrow keys           | Nudge 1px       |
| `Shift` + Arrow keys | Nudge 10px      |
| `Shift` + `H`        | Flip horizontal |
| `Shift` + `V`        | Flip vertical   |

## Stacking Order

| Shortcut                 | Action         |
| ------------------------ | -------------- |
| `Cmd/Ctrl` + `]`         | Move forward   |
| `Cmd/Ctrl` + `[`         | Move backward  |
| `Cmd/Ctrl` + `Alt` + `]` | Bring to front |
| `Cmd/Ctrl` + `Alt` + `[` | Send to back   |

## Grouping

| Shortcut                   | Action        |
| -------------------------- | ------------- |
| `Cmd/Ctrl` + `G`           | Group         |
| `Cmd/Ctrl` + `Shift` + `G` | Ungroup       |
| `Cmd/Ctrl` + `Alt` + `G`   | Wrap in frame |

## View

| Shortcut         | Action            |
| ---------------- | ----------------- |
| `Cmd/Ctrl` + `+` | Zoom in           |
| `Cmd/Ctrl` + `-` | Zoom out          |
| `Shift` + `1`    | Fit all           |
| `Shift` + `2`    | Zoom to selection |
| `Shift` + `R`    | Toggle rulers     |
| `Shift` + `C`    | Toggle comments   |

## Visibility and Lock

| Shortcut                   | Action            |
| -------------------------- | ----------------- |
| `Cmd/Ctrl` + `Shift` + `H` | Toggle visibility |
| `Cmd/Ctrl` + `Shift` + `L` | Toggle lock       |

## History

| Shortcut                   | Action       |
| -------------------------- | ------------ |
| `Cmd/Ctrl` + `Z`           | Undo         |
| `Cmd/Ctrl` + `Shift` + `Z` | Redo         |
| `Cmd/Ctrl` + `Alt` + `S`   | Save Version |

## Modifiers

| Modifier   | Effect                                                         |
| ---------- | -------------------------------------------------------------- |
| `Shift`    | Constrain aspect ratio while resizing                          |
| `Cmd/Ctrl` | Constrain to square while drawing, snap to 45/90 degree angles |
| `Alt`      | Temporarily switch to freehand drawing                         |
| `Space`    | Temporarily pan while using any tool                           |
