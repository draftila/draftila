# Layers & Pages

The left panel contains the layers tree and page management.

## Layers Panel

The layers panel displays all shapes on the current page in a hierarchical tree.

### Actions

- **Click** a layer to select it on the canvas
- **Cmd/Ctrl + click** to add/remove from selection
- **Double-click** a layer name to rename it
- **Drag and drop** layers to reorder them (changes z-order)
- **Right-click** for a context menu with cut, copy, paste, delete, and rename

### Visibility and Locking

Each layer has two toggles:

- **Eye icon** — Show or hide the shape (`Cmd/Ctrl` + `Shift` + `H`)
- **Lock icon** — Lock to prevent selection and editing (`Cmd/Ctrl` + `Shift` + `L`)

### Nesting

Frames and groups show an expand/collapse arrow. Click to reveal or hide children in the tree.

Double-click a group or frame on the canvas to enter it and interact directly with its children. Press `Escape` to exit back to the parent.

## Pages

Drafts can have multiple pages, each with its own independent layer hierarchy.

### Managing Pages

- **Add page** — Create a new page from the pages panel
- **Rename** — Double-click a page name to rename it
- **Delete** — Remove a page (at least one page must remain)
- **Switch** — Click a page to make it active

Only the active page is shown on the canvas at any time.
